# SGROAS – Sistema de Gestión de Recursos Operativos, Administrativos y de Seguridad

## Descripción

Aplicación web frontend desarrollada como Práctica Experimental de la Unidad I de la asignatura
de Desarrollo de Aplicaciones Web. SGROAS es una plataforma integral para la gestión de
cooperativas de transporte terrestre del Ecuador, que permite visualizar el directorio de
cooperativas, controlar flotas, registrar nuevas organizaciones y consultar estadísticas operativas.

## Tecnologías utilizadas

- HTML5 semántico con roles WAI-ARIA y accesibilidad WCAG 2.1 AA
- CSS3 con diseño Mobile-First, CSS Grid, Flexbox, variables CSS y modo oscuro nativo
- JavaScript ES6+ modular (api.js, ui.js, app.js) con consumo de API pública (JSONPlaceholder)
- Persistencia de datos con localStorage
